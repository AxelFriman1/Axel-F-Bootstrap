Local Bootstrap

Bootstrap används för att designa sidor. Bootstrap är bra för att det är enkelt att skapa responsiva hemsidor på ett enkelt sätt. Bootstrap gör allting responsivt automatisk utan att man behöver tänka på det som bilder och text. Bootstrap har förbestämda klasser med olika färg och placering som är enkla att använda sig av. Man kan även justera klasserna i efterhand om det är något man skulle vilja ändra med klassen. Bootstrap är baserat på ett rut-system.
