HTML

HTML står för HyperText Markup Language. HTML används för att bygga upp strukturen på en hemsida. Alla sidor använder sig av någon form av HTML för att strukturera upp deras hemsida. HTML är inte ett programmeringsspråk eftersom för att få kalla något ett programmeringsspråk så måste man kunna utföra kommandon vilket man inte kan i HTML. HTML är ett märkspråk.

Html element är element som strukturerar upp ditt html-dokument. T.ex. elementet html som omsluter hela dokumentet, elementet head där länkar till css, titeln på sidan och annat ligger eller elementet body där själva innehållet på ens sida ligger.
